What is frontend development?

Frontend is the development of the user interface and functions that work on the client side of a website or application. It's everything a user sees and interacts with when they open a web page. A frontend developer works with designers, programmers and UX analysts to create a convenient and popular product.

Career and salary of a front-end developer

Most often, the career of a frontend developer begins with a developer and tester. In the process of work, HTML + CSS is studied, then knowledge of JavaScript, libraries and frameworks is acquired. Then the basic concepts of building a server part are studied, adding tools in parallel, according to the chosen specialization. In addition, the skills of working with graphic editors and understanding the principles of UI / UX design and, of course, soft skills are developed.

In order to become a sought-after and experienced specialist in the field of front-end development, it is necessary to master the following technologies:

  • HTML and CSS (including grids and CSS frameworks, W3C and WHATWG specifications, HTML5 / CSS3 Polyfills);
  • Work freely with JavaScript;
  • Know popular libraries and frameworks: React.js, jQuery, Angular.JS, Redux, Backbone.js
  • CSS preprocessors (Sass, Less, Stylus, etc.);
  • Popular CMS (WordPress, Drupal, Joomla, etc.);
  • OOCSS / BEM / SMACSS;
  • ECMAScript 6;
  • DOM;
  • Understand the principles of Backend construction (Node.js, PHP, Ruby, .NET, etc.);
  • JavaScript transpilers (Babel);
  • Version control tools (Git, GitHub, CVS, etc.);
  • Databases and query languages (SQL, MySql, NoSQL, MongoDB and so on).
